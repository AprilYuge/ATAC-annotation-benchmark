# ATAC-annotation-benchmark

This reporsity contains scripts for reproducing results in two manuscripts related to benchmarking methods for cross-omics label transfer from single-cell RNA to ATAC data.

1. “*Benchmarking automated cell type annotation tools for single-cell ATAC-seq data*”. Published in *Frontiers in Genetics* in 2022. Folder *Frontiers_in_Genetics*.
2. "*A Comprehensive Benchmarking Study on Computational Tools for Cross-omics Label Transfer from Single-cell RNA to ATAC Data*". Published in *G3: Genes|Genomes|Genetics* in 2026. Folder *G3*.
