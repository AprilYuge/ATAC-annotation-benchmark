# A Comprehensive Benchmarking Study on Computational Tools for Cross-omics Label Transfer from Single-cell RNA to ATAC Data

This reporsity contains scripts and notebooks for reproducing results in the manuscript “*A Comprehensive Benchmarking Study on Computational Tools for Cross-omics Label Transfer from Single-cell RNA to ATAC Data*” published in *G3: Genes|Genomes|Genetics*. 

The *scripts/* directory includes implementations of all computational methods benchmarked in the study. 

The *metrics/* directory contains a Python module with functions for computing evaluation metrics, as well as a Jupyter notebook used to generate benchmarking results and visualization figures.
